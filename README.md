# recorrido 



**Nombre:**
**“Archivo felino: historia de los gatos y los humanos”**

**Concepto:** un recorrido por “salas” o “espacios” que representan momentos históricos (reales o ligeramente ficcionalizados), donde el usuario va avanzando como si explorara una línea del tiempo viva. Web interactiva sobre la relación entre los gatos y los humanos a través de la historia.
